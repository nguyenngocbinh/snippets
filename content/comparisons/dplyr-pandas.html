So sánh dplyr (R) và pandas (Python) - Hướng dẫn thao tác dữ liệu

Bảng so sánh chi tiết các thao tác xử lý dữ liệu giữa dplyr (R) và pandas (Python):

🔧 Các phương thức xử lý Data Frame

1. 📋 Thao tác với Hàng (Rows)

Chức năng dplyr (R) pandas (Python)
Arrange (Sắp xếp) arrange(df, col) df.sort_values(by='col', ascending=True)
Distinct (Loại bỏ trùng) distinct(df, col) df.drop_duplicates(subset='col')
Filter (Lọc dữ liệu) filter(df, condition) df[df['condition']]
Slice (Cắt dữ liệu) slice(df, rows) df.iloc[rows]

2. 📊 Thao tác với Cột (Columns)

Chức năng dplyr (R) pandas (Python)
Glimpse (Xem tổng quan) glimpse(df) df.info()
Mutate (Tạo cột mới) mutate(df, new_col = func(old_col)) df['new_col'] = df['old_col'].apply(func)
Pull (Trích xuất cột) pull(df, col) df['col']
Rename (Đổi tên cột) rename(df, new_col_name = old_col_name) df.rename(columns={'old_col_name': 'new_col_name'}, inplace=True)
Select (Chọn cột) select(df, col1, col2, ...) df[['col1', 'col2', ...]]

3. 👥 Thao tác với Nhóm (Groups)

Chức năng dplyr (R) pandas (Python)
Group By (Nhóm theo) group_by(df, col) df.groupby('col')
Summarise (Tổng hợp) summarise(df, new_col = func(col)) df.agg({'col': func})

4. 🔗 Thao tác với Data Frames

Chức năng dplyr (R) pandas (Python)
bind_cols (Nối cột) bind_cols(df1, df2) pd.concat([df1, df2], axis=1)
bind_rows (Nối hàng) bind_rows(df1, df2) pd.concat([df1, df2], axis=0)
Inner Join (Nối trong) inner_join(df1, df2, by = "key_column") pd.merge(df1, df2, on='key_column', how='inner')
Left Join (Nối trái) left_join(df1, df2, by = "key_column") pd.merge(df1, df2, on='key_column', how='left')
Right Join (Nối phải) right_join(df1, df2, by = "key_column") pd.merge(df1, df2, on='key_column', how='right')
Full Join (Nối đầy đủ) full_join(df1, df2, by = "key_column") pd.merge(df1, df2, on='key_column', how='outer')
Semi Join (Nối bán) semi_join(df1, df2, by = "key_column") Không có sẵn; có thể dùng merge và isin để đạt hiệu quả tương tự.
Anti Join (Nối ngược) anti_join(df1, df2, by = "key_column") Không có sẵn; có thể dùng merge và isin để đạt hiệu quả tương tự.

5. ⚡ Hàm xử lý Vector

Chức năng dplyr (R) pandas (Python)
if_else (Điều kiện) mutate(df, new_col = if_else(condition, true_val, false_val)) df['new_col'] = np.where(condition, true_val, false_val)
na_if (Thay NA) mutate(df, col = na_if(col, value)) df['col'].replace(value, np.nan)
n_distinct (Đếm riêng) n_distinct(df, col) df['col'].nunique()
sample_n (Mẫu số) sample_n(df, n) df.sample(n=n)
sample_frac (Mẫu %) sample_frac(df, fraction) df.sample(frac=fraction)
case_when (Nhiều điều kiện) mutate(df, new_col = case_when(condition1 ~ value1, condition2 ~ value2, ...)) df['new_col'] = np.select([condition1, condition2, ...], [value1, value2, ...], default=default_value)
cummean (Trung bình tích lũy) mutate(df, new_col = cummean(col)) df['new_col'] = df['col'].expanding().mean()
row_number (Số thứ tự) mutate(df, row_num = row_number()) df['row_num'] = range(1, len(df)+1)
min_rank (Xếp hạng min) mutate(df, rank = min_rank(col)) df['rank'] = df['col'].rank(method='min')
dense_rank (Xếp hạng dày) mutate(df, rank = dense_rank(col)) df['rank'] = df['col'].rank(method='dense')

💡 Lưu ý: Mặc dù nhiều chức năng có sẵn trực tiếp trong cả dplyr và pandas, một số có thể cần biến thể nhỏ hoặc hàm tùy chỉnh để đạt được kết quả tương tự.
